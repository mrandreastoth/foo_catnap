# RESTful API for foobar2000

Default port: 20402 = 0x4FB2

GET /playlists
GET /playlists/active
GET /playlists/playing
GET /playlists/:playlist
GET /playlists/active/items
GET /playlists/playing/items
GET /playlists/:playlist/items
